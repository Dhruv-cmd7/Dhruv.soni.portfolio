# Portfolio Site

A static portfolio site with a cleaner, professional project layout.

## Structure

- `index.html`, `about.html`, `services.html`, `work.html`, `contact.html` - main site entry points
- `assets/css/style.css` - shared stylesheet
- `assets/js/` - shared JavaScript utilities
- `assets/` - images and audio assets
- `scraped/` - archived Framer export pages and related output
- `tools/inspect/` - developer scripts for checking HTML/CSS structure
- `local-server.js` - lightweight local static server

## Run locally

```bash
node local-server.js
```

Then open `http://127.0.0.1:8000/` in your browser.
